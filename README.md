# Mini-Project-2

Pada mini project kali ini, saya membuat REST API menggunakan expressJS untuk mengimplementasikan system reseller order pada suatu toko, dimana terdapat 3 role user yaitu owner, reseller, dan buyer. Owner adalah selaku pemilik usaha, reseller adalah orang yang ingin menjual ulang product yang dimiliki oleh owner, dan buyer adalah orang yang membeli prudct yang dijual reseller. 

Konsep dari system ini yaitu owner membuat product lalu reseller "mengambil" product tersebut untuk dijual kembali dengan mendapatkan komisi 10% dari owner, sementara buyer hanya melihat product yang dimiliki atau yang sudah diambil reseller untuk dijual kembali, ketika buyer membeli peroduct tersebut maka harga akan dipecah 10% untuk reseller dan 90% untuk owner.
